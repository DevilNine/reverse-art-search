# Security Policy

## Supported Versions

The Reverse Artist Search platform currently supports the latest live version.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do not open a public issue. 
Instead, reach out privately.

1. This repository contains the frontend logic.
2. The backend is protected under the DevilNine Security Protocol (which includes strict SSRF checks and CORS dynamic validation).

If you believe you have found a bypass or API token leak, please contact the repository owner (DevilNine) directly.
