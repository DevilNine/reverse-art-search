## 🌟 Features

* **Multi-Source Power**: Simultaneously searches through engines like Google Lens, SauceNAO, IQDB, and Yandex to guarantee the highest match rate.
* **Premium UX/UI**: Designed with a sleek, dark-glassmorphism aesthetic. Vibrant red accents and silky-smooth micro-animations provide an elite user experience.
* **Smart History**: Keep track of previous searches locally. The app elegantly displays your uploaded image alongside past results.
* **Blazing Fast**: Optimized Single Page Application (SPA) built for maximum performance and deployed seamlessly via Cloudflare Pages.
* **Mobile Ready**: Fully responsive interface with accessible 44x44px touch targets.

## 🛠️ Technical Details

This repository contains the **production build** of the Reverse Artist Search frontend application. It is heavily obfuscated and minified (using esbuild) to protect the core intellectual property while allowing the project to be freely hosted and demonstrated.

* **Architecture**: React + Vite (SPA)
* **Hosting**: Configured for Cloudflare Pages with built-in security headers (`_headers`) and SPA routing rules (`_redirects`).
* **Styling**: Vanilla CSS utilizing custom properties for theme management, ensuring zero-dependency overhead.
